# Mini YouTube App

> [View Demo](https://mini-youtube-app.netlify.app/)
>
> This is a simple mini YouTube Clone using https://developers.google.com/youtube/v3

## Technologies

> 1. `React Hooks`
> 2. `Material UI`
> 3. `YouTube v3 API`

## Instructions:

> 1. `Clone the Repo`
>
> 2. `npm install`
>
> 3. `npm start`
